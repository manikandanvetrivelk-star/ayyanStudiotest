# Footer Gallery Images

Add your Indian & Tamil Nadu wedding photos here.

## Instructions

1. **Place your images in this folder** with the following naming convention:
   - `wedding-1.jpg`
   - `wedding-2.jpg`
   - `wedding-3.jpg`
   - `wedding-4.jpg`
   - `wedding-5.jpg`
   - `wedding-6.jpg`

2. **Image Requirements:**
   - Format: JPG, PNG, or WebP
   - Size: Recommended 400x400px or larger
   - Quality: High resolution (for best gallery appearance)
   - Aspect Ratio: Square (1:1) works best

3. **Example Setup:**
   ```
   public/
   └── images/
       ├── wedding-1.jpg
       ├── wedding-2.jpg
       ├── wedding-3.jpg
       ├── wedding-4.jpg
       ├── wedding-5.jpg
       └── wedding-6.jpg
   ```

4. **After adding images:**
   - Refresh your browser
   - Images will automatically appear in the footer gallery
   - They will display with grayscale filter and scale effect on hover

## Image Tips

- Use high-quality wedding photography
- Ensure consistent lighting and color treatment
- Square crop (1:1) recommended for uniform gallery
- File size should be under 1MB for optimal performance

The Footer component automatically references images from `/images/` folder.
